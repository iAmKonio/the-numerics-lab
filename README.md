# The Numerics Lab

Numerical methods from the implementation point of view.

The Numerics Lab is a practical code-and-methods lab and a companion repository to the [Konio Engineering Lab](https://konioenglab.com/numerics-lab). 

## Purpose
This repository serves as a collection of:
- Small CFD programs
- Implementation notes
- Discretization experiments
- Verification cases
- Reproducible examples

## Major Categories

1. **Finite Difference Methods**
2. **Finite Volume Methods**
3. **Finite Element Methods**
4. **Discontinuous Galerkin Methods & DGSEM**
5. **Lattice Boltzmann Methods**

### Examples
Examples are developed as small, readable programs.

*Suggested initial example path:*

## Contributing
This is a repository of public educational and research code.

- No confidential project data should be included.
